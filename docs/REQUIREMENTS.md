# Dustman 需求分析文档

> Windows 平台桌面端垃圾清理工具。技术栈 Flutter (Windows Desktop) + Dart。

---

## 1. 项目背景

Windows 系统在日常使用中会产生大量难以手动清理的临时数据：浏览器/系统缓存、
更新残留、缩略图数据库、卸载残留文件夹与注册表项等。这些数据会逐渐占满 SSD
分区、降低系统响应速度，且对普通用户而言难以辨识哪些可以安全删除。

Dustman 的目标是提供一款**安全、可视化、可解释**的 Windows 桌面清理工具，
让用户在不破坏系统的前提下回收磁盘空间。

## 2. 目标用户

| 用户画像 | 需求关注点 |
|---|---|
| 普通 PC 用户 | 一键扫描 / 一键清理、明确知道删了什么 |
| 开发者 / 重度用户 | 自定义路径扫描、大文件定位、卸载残留清理 |
| IT 运维 | 可移植绿色版、可命令行调用（未来阶段） |

## 3. 功能需求

### 3.1 MVP 范围（v0.1 - 本期交付）

| 模块 | 功能点 | 优先级 |
|---|---|:-:|
| 垃圾扫描 | 用户 / 系统临时文件 (`%TEMP%`、`C:\Windows\Temp`) | P0 |
| 垃圾扫描 | 主流浏览器缓存（Chrome / Edge / Firefox） | P0 |
| 垃圾扫描 | Windows 日志与崩溃转储（`C:\Windows\Logs`、`*.dmp`） | P0 |
| 垃圾扫描 | 缩略图缓存（`thumbcache_*.db`） | P0 |
| 垃圾扫描 | DNS 缓存（`ipconfig /flushdns`） | P1 |
| 垃圾扫描 | 回收站统计与清空（Win32 `SHQueryRecycleBin` / `SHEmptyRecycleBin`） | P0 |
| 清理执行 | 分类多选 / 单项预览 / 二次确认 | P0 |
| 清理执行 | 实时进度（已扫描路径 / 找到大小） | P0 |
| 清理执行 | 清理结果汇总（释放空间、失败项） | P0 |
| 通用 | Material 3 主题、明暗切换、中英文 i18n | P1 |

### 3.2 第二阶段（v0.2）✅ 已交付

- 卸载残留扫描：扫描 `Program Files` / `AppData` / `ProgramData` / `HKLM\Software`
  下与已卸载程序匹配的孤儿目录与注册表项；
- 大文件查找器：按目录递归扫描，按大小排序，支持过滤后缀；
- 重复文件检测：基于 SHA1 + 文件大小预筛；
- 启动项管理：列出 `HKCU` / `HKLM` 下 `Run` / `RunOnce` 及启动文件夹的 `.lnk`，支持禁用。

### 3.3 第三阶段（v0.3+）

- 磁盘空间可视化（TreeMap，类似 WinDirStat）；
- 已安装程序列表与一键卸载；
- 清理计划（每周自动扫描提醒）；
- 命令行模式（`dustman.exe scan --json`）；
- 多语言扩展、绿色版打包（zip 分发 + portable 设置）。

## 4. 非功能需求

### 4.1 安全性 (P0)
- **白名单保护**：扫描器内置“受保护路径”集合，绝不删除 `Windows`、`System32`、
  当前用户文档目录、桌面等敏感目录；
- **二次确认**：所有清理操作需用户主动勾选 + 弹窗确认；
- **可撤销策略**：默认走系统回收站（Win32 `SHFileOperation` + `FOF_ALLOWUNDO`），
  仅在用户显式选择“彻底删除”时绕过；
- **管理员权限提示**：检测到需要权限的路径（如 `C:\Windows\Temp`）时给出
  UAC 提升指引，而非静默失败；
- **结果可追溯**：清理日志写入 `%APPDATA%\Dustman\logs\YYYY-MM-DD.log`。

### 4.2 性能
- 扫描在 `Isolate` 中执行，不阻塞 UI；
- 单分类扫描可独立中断；
- 文件枚举使用 `Directory.list(followLinks: false)` 流式处理，避免一次性加载。

### 4.3 易用性
- 首屏即可看到“总可清理空间”估算；
- 每个分类提供“是什么 / 为什么安全删除”提示；
- 全程不打扰：默认不联网、无广告、无强制注册。

### 4.4 兼容性
- 目标平台 Windows 10 1809+ / Windows 11；
- 架构 x64（aarch64 后续阶段评估）；
- Flutter SDK ≥ 3.22，Dart ≥ 3.4。

## 5. 范围之外（Non-Goals）

- 不做注册表"优化"、不做"系统加速"、不做"内存清理"等噱头功能；
- 不联网上传任何文件/路径数据；
- 不替代杀毒软件，不处理恶意软件清除；
- 本期不支持 macOS / Linux（架构层预留扩展点）。

## 6. 验收标准（MVP）

1. 在干净的 Windows 11 虚拟机上 `flutter run -d windows` 可启动；
2. 首页可显示 6 个内置分类的扫描结果与累计大小；
3. 勾选任意分类后点击“清理”，能成功释放空间，且 UI 数字归零；
4. 误删受保护路径下的文件应被白名单拦截，并在日志中记录被拦截的路径；
5. 关闭应用并重启后，主题与窗口尺寸偏好被记住。
